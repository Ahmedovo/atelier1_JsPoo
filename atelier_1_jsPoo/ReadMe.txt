EXERCICE 1 : 
    Aperçu :
Ce code JavaScript fournit un système simple de gestion de l'inventaire pour les voitures. 
Il définit des constructeurs pour différents modèles de voitures, permet la création d'instances de ces voitures,
 et permet le tri et l'affichage de l'inventaire en fonction de l'année de fabrication.
   
    Structure du Code :
-Définition du constructeur de base Voiture.
-Initialisation du tableau listVoitures.
-Définition des sous-classes Hyundai et Ford qui héritent des propriétés de Voiture.
-Remplissage de l'inventaire avec des instances de voitures Hyundai et Ford.
-Tri de l'inventaire par année de fabrication.
-Affichage de l'inventaire trié.

EXERCICE 4 : 
    Aperçu
Cette page HTML représente une interface utilisateur pour un blog.
 Les utilisateurs peuvent visualiser les articles existants, ajouter de nouveaux articles et afficher les détails des articles.
    
    Structure du Code : 
-Code
En-tête HTML: Définit les métadonnées de base et inclut le lien vers Bootstrap CSS pour la mise en page.

Barre de navigation: Contient le nom du blog et des liens vers différentes sections du site.

-Contenu principal:

Formulaire modal d'ajout d'article: Un formulaire modal pour permettre aux utilisateurs d'ajouter de nouveaux articles avec un titre et une description.
Liste des articles: Une section pour afficher les articles existants.
Scripts JavaScript:

Définit les classes User, Post, et Blog pour gérer les utilisateurs, les articles et le blog.
Crée une instance de Blog.
Contient des fonctions pour ajouter un nouvel article, afficher tous les articles, et gérer les interactions avec les boutons et les formulaires.

-Fonctionnement
L'utilisateur peut ajouter un nouvel article en remplissant le formulaire modal et en cliquant sur le bouton "POST".
Les articles sont stockés dans un tableau en mémoire et affichés dans la section des articles.
La page utilise Bootstrap pour le style et l'interactivité.
